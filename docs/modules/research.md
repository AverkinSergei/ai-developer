# Research-контур

## `research`

Контур исследования для задач `task_type=research`. `run_research` готовит markdown-отчёт
(выводы, риски, варианты решений, рекомендация) и публикует его в задачу Битрикс24 с меткой
`[AI_RESEARCH]`. **Ветка и MR не создаются.**

Если по итогам исследования нужна реализация — создаётся новая code-задача или меняется
`task_type` после подтверждения человеком.
